# Чеклист отладки ChesscomSkins v2

## Как проверить работу расширения

### 1. Установка
1. Открой `chrome://extensions/`
2. Включи "Режим разработчика"
3. Удали старую версию расширения
4. Нажми "Загрузить распакованное" → выбери папку ChesscomSkins
5. Перезагрузи страницу chess.com

### 2. Открытие консоли
- F12 → Console
- Фильтруй по `[ChesscomSkins` чтобы видеть только наши логи

### 3. Проверка функций

#### Главный переключатель
- [ ] При включении в консоли: `[ChesscomSkins] Storage data: {enabled: true, ...}`
- [ ] При выключении все эффекты исчезают

#### Скины
- [ ] Клик на скин → `[ChesscomSkins] Skin applied: set2`
- [ ] Фигуры меняют внешний вид

#### Glow эффекты
- [ ] Клик на Ember → `[ChesscomSkins] Effect applied: native-ember`
- [ ] Фигуры получают оранжевое свечение
- [ ] Frost → голубое свечение
- [ ] Neon → фиолетовое свечение

#### Glow Intensity слайдер
- [ ] При движении: `[ChesscomSkins] Glow intensity set to: X`
- [ ] Свечение увеличивается/уменьшается

#### Стили доски
- [ ] Клик на Marble → `[ChesscomSkins] Board style applied: marble`
- [ ] Клетки меняют цвет
- [ ] Neon → тёмная доска с голубым свечением

#### Visual Effects (Particles/Cracks)
- [ ] При включении Particles: `[ChesscomSkins:Animations] Storage: {particlesEnabled: true}`
- [ ] При ходе появляются частицы
- [ ] При включении Cracks появляются трещины

## Известные ограничения

1. **Частицы и трещины** работают только во время игры когда есть ходы
2. **Стили доски** работают только на страницах с реальной шахматной доской
3. После изменения настроек может потребоваться перезагрузка страницы

## Ошибки и решения

### "Storage data: {}" пустой
Расширение загружается впервые. Включи главный переключатель.

### Скин не применяется
Проверь что файлы в `assets/Set2/` существуют.

### Эффекты не видны
1. Убедись что скин включен
2. Попробуй другой эффект
3. Увеличь Glow Intensity
