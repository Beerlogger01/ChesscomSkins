# Как добавить новый набор скинов

Система сейчас работает динамически! Вот что нужно сделать для добавления новых скинов:

## Шаг 1: Подготовьте файлы скинов

Создайте новую папку в `assets/` с названием вашего набора. Например, `assets/MyNewSkin/`

Папка должна содержать 12 PNG файлов с фигурами:
```
MyNewSkin/
  ├── wk.png  (белый король)
  ├── wq.png  (белая королева/ферзь)
  ├── wr.png  (белая ладья)
  ├── wb.png  (белый слон)
  ├── wn.png  (белый конь)
  ├── wp.png  (белая пешка)
  ├── bk.png  (чёрный король)
  ├── bq.png  (чёрная королева/ферзь)
  ├── br.png  (чёрная ладья)
  ├── bb.png  (чёрный слон)
  ├── bn.png  (чёрный конь)
  └── bp.png  (чёрная пешка)
```

## Шаг 2: Добавьте запись в конфиг

Отредактируйте файл `assets/skins.json` и добавьте новый набор:

```json
{
  "skins": [
    {
      "id": "set2",
      "name": "Festive Classic",
      "folder": "Set2",
      "description": "Beautiful festive-themed chess pieces",
      "previewPiece": "wk"
    },
    {
      "id": "mynewskin",
      "name": "My New Skin",
      "folder": "MyNewSkin",
      "description": "Описание вашего набора",
      "previewPiece": "wk"
    }
  ]
}
```

### Параметры конфига:
- **id**: уникальный идентификатор (только латинские буквы и цифры, без пробелов)
- **name**: красивое название для отображения в UI
- **folder**: название папки в `assets/` (должна совпадать с названием папки)
- **description**: описание набора
- **previewPiece**: какая фигура будет показана в превью эффектов (по умолчанию "wk")

## Шаг 3: Готово!

Расширение автоматически:
1. Загрузит конфиг при открытии popup
2. Создаст новый элемент в интерфейсе
3. Будет готово использовать ваш новый набор скинов

## Примечания

- Размер изображений фигур не критичен, они будут масштабированы
- Формат: PNG с прозрачностью (для лучшего качества)
- После добавления в `skins.json` нужно перезагрузить расширение (чтобы новый конфиг был загружен)

## Пример структуры папок

```
ChesscomSkins/
├── assets/
│   ├── skins.json          ← отредактируйте этот файл
│   ├── Set2/               ← существующий набор
│   │   ├── wk.png
│   │   ├── wq.png
│   │   ├── ... (остальные фигуры)
│   ├── MyNewSkin/          ← ваш новый набор
│   │   ├── wk.png
│   │   ├── wq.png
│   │   └── ... (остальные фигуры)
├── Popup/
├── manifest.json
└── ... (остальные файлы)
```
